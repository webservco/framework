# DateHelper

---

## Usage

```php
$result = \WebServCo\Framework\Helpers\DateHelper::method($arguments);
```

---

## Static methods

### `format(string $date, string $format = 'Y-m-d')`

Format a date.


### `validate(string $date, string $format = 'Y-m-d')`

Validate an already formatted date.
